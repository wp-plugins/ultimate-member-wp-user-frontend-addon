Integration of \"Ultimate Member\" + \"WP User Fronted\" in user profiles

== Description ==
The plugin was integrat \"Ultimate Member\" with \"WP User Fronted\".
It\'s add in user profile additional pages to add posts and list of posts (with edit options).
We can select in what roles show additional pages.

== Installation ==
Install Ultimate Member
Install WP User Fronted
Configure edit page for WP User Fronted

Install our plugin and select roles where we want to display additional pages in profile.
